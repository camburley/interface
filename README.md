# Burley.ai

**AI agent engineering studio** — we build the infrastructure, APIs, and tools that make AI agents actually work in production.

Specializing in:
- Agent-native APIs and endpoints
- LangGraph-powered workflow orchestration
- Agent-to-agent communication protocols
- Production AI integrations (OpenAI, Claude, Gemini)

## Products & Open Source

### [Supermarket Puzzle](https://github.com/camburley/shopping-game)
Grocery intelligence API for AI agents. Real-time pricing, product data, and smart routing for agentic shopping workflows.

### [Realtime Voice Agents](https://github.com/camburley/twilio-realtime-agent)
Open-source voice agent SDK — 2k+ GitHub stars. Twilio + LangGraph + streaming TTS.

### [Prediction Engine](https://github.com/camburley/prediction-engine)
Deterministic arbitrage signals for prediction markets. Guardrail-enforced, production-grade.

### [OpenClaw Agent Skills](https://github.com/openclaw/openclaw)
Agent skill framework for extensible AI assistants. Skills for Slack, GitHub, browser control, and more.
